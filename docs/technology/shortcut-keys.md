- 选中某段文本后按 ctrl + D 批量选中与选中文本相同的文本（大小写不敏感）
![ctrl+D](/public/shortcut-keys/ctrl+d.png)

- ctrl + U 为 ctrl + D 的逆操作，即取消选中刚才 ctrl + D 选中的文本

- 长按 alt 鼠标点击多个任意位置，可以在不同位置放置多个光标，便于一次性在多个不同的地方插入相同的文本
![alt+mouse](/public/shortcut-keys/alt+mouse.png)

- 长按 shift + alt 用鼠标分别点击首行以及末行同一列位置，可以在首行与末行之间的每行的同一位置都放置一个光标
![shift+alt+mouse](/public/shortcut-keys/shift+alt+mouse1.png)

- 长按 shift + alt 用鼠标分别点击首行以及末行不在同一列位置，可以将首行与末行之间的每一行的两列直接的部分都选中
![shift+alt+mouse](/public/shortcut-keys/shift+alt+mouse2.png)

- alt + B 若当前文件为 html 文件，则使用默认浏览器打开当前文件

- alt + Z 将当前文件文本自动换行，无需横向滚动看完整文本

- shift + alt + up/down 方向键，将光标当前所在行或者当前选中的所有行整体复制一份到当前位置的上/下方

- shift + ctrl + K 删除光标所在行或当前选中的所有行

- ctrl + Enter 将光标移到当前所在行的下一行开头（无需将光标移动到当前行末尾再按 Enter）

- ctrl + shift + Enter 将光标移到当前所在行的上一行开头

- alt + up/down 方向键，将光标当前所在行或者当前选中的所有行整体上移/下移一行

- /** + Enter 文档注释（doc 注解）

- ctrl + up/down 保持光标位置不动，文件整体上下滚动

- shift + Tab 回撤一个 Tab 距离（貌似和 Backspace 没区别？）

- ctrl + ` 打开 VSCode 内置终端

- shift + ctrl + ` 打开 VSCode 新终端

- ctrl + O 用 VSCode 打开某个文件

- ctrl + O + K 用 VSCode 打开某个文件夹

- ctrl + R 打开最近打开过的项目列表

- ctrl + [ 将当前行向前缩进

- ctrl + ] 将当前行向后缩进

- ctrl + Backspace 一次性删除两个空格之间的文本

- ctrl + 双击某个文件 新开一个 tab 打开当前文件
